Welcome to Stellar Web

Build your own network of stars in space!

Getting Started:

  • Click anywhere on the canvas to spawn star nodes
  • Use the sliders (gear icon ⚙️) to adjust:
    • Node count, connectivity, speed, size
    • Trail effects and mouse gravity
  • Move your mouse to attract nearby stars
  • Hover over stars to see their connections
  • Watch as stars bounce off edges and form networks

Tips:

  • Start by clicking to create 5-10 stars
  • Adjust connectivity to see different patterns
  • Try the Stats tab to see network metrics
×

How to Use Stellar Web

  • Click anywhere to spawn a new node
  • Hover over a node to highlight its connections
  • Move your cursor to attract nearby nodes (gravitational pull)
  • Use sliders to adjust network parameters
  • Pause button freezes all movement
  • Reset button clears and regenerates the network
  • Stats tab shows real-time network metrics
