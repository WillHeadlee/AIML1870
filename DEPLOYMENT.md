## Deploy to GitHub Pages

### Option 1: Enable GitHub Pages (Recommended)

1. Go to your repository: https://github.com/WillHeadlee/AIML1870
2. Click on **Settings** tab
3. Scroll down to **Pages** in the left sidebar
4. Under **Source**, select **Deploy from a branch**
5. Select **main** branch and **/ (root)** folder
6. Click **Save**
7. Wait a few minutes for deployment

Your site will be available at:
**https://willheadlee.github.io/AIML1870/stellar-web/**

### Option 2: Local Testing

Simply open the file directly in your browser:
```
stellar-web/index.html
```
